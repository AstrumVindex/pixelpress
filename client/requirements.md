## Packages
browser-image-compression | Client-side image compression library
react-dropzone | Drag and drop file upload handling
framer-motion | Smooth UI transitions and animations
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge Tailwind classes efficiently

## Notes
- This is a client-side only app (PWA style).
- All compression happens in the browser.
- No backend endpoints are used for image processing.
- Images should be processed using web workers where possible (handled by the library).
